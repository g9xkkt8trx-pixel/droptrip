# DROPTRIP Production Smoke Test

対象: `https://droptrip.vercel.app` / 正式版 `v1.0.0`

## 基本画面

- [ ] トップ画面が表示される
- [ ] 「沖縄」「箱根」などで検索し、詳細画面へ移動できる
- [ ] hero画像、映えスポット、お気に入り、Google Mapsボタンを確認する
- [ ] 戻る・再読み込み後も主要画面が壊れない
- [ ] 幅390pxで横スクロールや主要操作の欠落がない

## 外部API

- [ ] 出発地を入力し、移動時間取得が成功または安全なエラー表示になる
- [ ] AIプランを生成し、日別カードで表示される。生JSONを表示しない
- [ ] AIの1日1回制限、キャッシュ再表示、失敗時に回数を消費しない挙動を確認する
- [ ] Vercel Network/Functionログで想定外の4xx/5xxがない

## 公開・案内

- [ ] `/robots.txt` が200で表示される
- [ ] `/terms.html`、`/privacy.html`、`/contact.html` が200で表示される
- [ ] フッターの利用規約、プライバシー、お問い合わせ、バージョン表示を確認する
- [ ] フッターから`/contact.html`へ到達でき、設定済みの場合だけ外部お問い合わせフォームが新しいタブで開く
- [ ] お問い合わせページ・法務ページ・一般画面に開発者個人やソース管理サービスの情報が出ていない
- [ ] Consoleエラー、Networkエラー、画像404がない

## PWA

- [ ] `/manifest.webmanifest`、`/service-worker.js`、`/offline.html`、PWAアイコンが200で表示される
- [ ] AndroidまたはPCでインストール後に単独アプリ表示で開ける
- [ ] iPhoneではSafariの共有メニューからホーム画面へ追加できる
- [ ] オフライン時に安全な案内が表示され、AI・移動時間がキャッシュされない
- [ ] 新しいデプロイ後の「更新する」操作で最新画面へ更新できる

## 障害対応確認

- [ ] `ENABLE_AI_PLAN=false` のとき、AIだけがメンテナンス表示になる
- [ ] `ENABLE_ROUTE_TIME=false` のとき、移動時間だけが安全に停止する
- [ ] 復旧後に対象機能だけを再確認する
